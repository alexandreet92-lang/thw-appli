# Contenu détaillé des 16 thèmes pour le site vitrine

## Contexte
Le site vitrine THW Coaching (https://thw-appli.vercel.app) s'appuie sur 16 thèmes
(cf. `FEATURES_USER.md`). Chaque thème aura sa page dédiée, créée plus tard par
Claude Design. Ici, on produit uniquement les **sources de contenu** — pas le site.

## Livrable
`CONTENU_SITE_THEMES.md` à la racine : Markdown long, navigable, self-contained.
- Sommaire cliquable (ancres) en tête.
- 16 sections détaillées (une par thème).
- Liens d'ancres entre thèmes liés.
- Lien « ↑ Retour au sommaire » en fin de chaque section.

## Structure par thème (minimum)
- **Vue d'ensemble** : tagline (10-15 mots), description (2-3 phrases), 3-5 bénéfices, profil ciblé.
- **Sous-thèmes** (tous ceux identifiés) — chacun : titre, tagline, description narrative
  (3-6 phrases), détails/valeurs, exemple concret, visuel suggéré, citation à mettre en avant.
- **Cas d'usage** (2-3 scénarios).
- **Limites par plan** (tableau Premium/Pro/Expert si applicable).
- **Thèmes liés** (ancres).
- **CTA suggéré**.
- **Notes pour Claude Design** (à mettre en avant / à éviter / ambiance visuelle).

## Règles
Ne décrire que le réel. Ton premium, direct, tutoiement, concret/chiffré, sans jargon
technique (pas de tables/fichiers/routes). Cohérence : prix 14/26/49 €, modèles
Hermès/Athéna/Zeus, plans Premium/Pro/Expert, multiplicateurs ×1/×3/×8.

## Livrable final
Fichier + commit « docs: detailed content for 16 site themes » + résumé (nb lignes,
thèmes les plus riches, thèmes les plus minces).
