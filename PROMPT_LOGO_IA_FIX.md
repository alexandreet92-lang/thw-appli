Vérifier aussi tous les autres endroits où ces modèles sont affichés

d'entraînement / Zeus", "Identifier mes points faibles / Athèna", 
etc.) — remplacer aussi ces petites icônes par les nouveaux logos.

"+" à côté de "CONVERSATIONS" sont actuellement violets.
Les passer en bleu cohérent avec le reste de l'app.

(le bleu/cyan accent principal de l'app).
