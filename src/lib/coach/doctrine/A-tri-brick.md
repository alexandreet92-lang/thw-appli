# A — TRIATHLON · BRICK (ENCHAÎNEMENTS)

> **Statut : DOCUMENT MÉTHODE (compétence spécifique).** Choix/placement = **B1 §3** + **A-tri-periodisation-multisport** ; pacing par format = **B6 §4** ; zones = **B4** ; charge = **B5** ; nutrition = **B9**.
> **Mapping charge :** un brick = double charge SM + SN spécifique des deux sports enchaînés (B5 §8) 〔HYP〕.

---

## 1. PRINCIPE
Le **brick** = enchaîner deux disciplines sans pause (surtout **vélo→course**, parfois **nage→vélo**) pour entraîner ce que le triathlon a d'unique : **courir avec les « jambes de coton »** après le vélo, **pacer le vélo pour pouvoir courir**, et **rôder les transitions (T1/T2)**. C'est une **compétence neuromusculaire et mentale** que ni le vélo seul ni la course seule ne développent.

---

## 2. POUR QUI / QUAND (B1 §3, B6)
**Indispensable** pour tout triathlète, surtout en **phase spécifique** (build/pré-course). Le **format** dicte le type de brick (B6 §4).
> `SI loin de l'échéance (base) → ALORS bricks rares/courts (l'adaptation est rapide, inutile d'en abuser tôt). SI phase spécifique → ALORS bricks réguliers à intensité de course.`
> `SI temps très limité → ALORS le brick fait « d'une pierre deux coups » (efficace), mais reste coûteux en fatigue : pas plus d'1-2/sem.`

---

## 3. TYPES DE BRICK & CALIBRAGE (B6 §4)
| Format visé | Brick type | Intensités |
|---|---|---|
| **Sprint / Olympique** | Vélo intensif court → course rapide | Vélo proche seuil → course @ allure course (rapide) |
| **70.3** | Vélo long tempo → course tenue | Vélo IF ~0,80-0,85 FTP → course @ allure semi cible |
| **Ironman** | Vélo long Z2-bas Z3 → course modérée | **Vélo IF ~0,65-0,72** → course @ allure marathon, **avec ravitaillement** (B9) |

- **Progression :** commencer court (vélo + 10-15 min course), allonger la fraction course vers la spécificité du format.
- **Transitions :** répéter T1/T2 (pose/chaussage, sortie, premières foulées) — temps gratuit en course.
- **Nage→vélo** (moins fréquent) : utile pour gérer la sortie d'eau et les premières minutes vélo (jambes/tête).

---

## 4. RÈGLE D'OR DU BRICK
> **Le brick valide le pacing vélo, pas le chrono vélo.** `SI on roule trop fort sur la partie vélo du brick → ALORS la course s'effondre : c'est précisément la leçon à intégrer (rouler à l'IF du format, B6 §4, pour pouvoir courir).`
> `SI Ironman → ALORS le brick teste AUSSI le plan nutrition (B9) : enchaîner course après vélo révèle la tolérance digestive sous fatigue. Ne jamais dissocier brick long et nutrition.`

---

## 5. PROGRESSION & CAS LIMITES
> `SI les premières foulées post-vélo sont catastrophiques → ALORS c'est normal et ça s'entraîne : multiplier les transitions courtes améliore vite la sensation.`
> `SI fatigue cumulée élevée (B5/B8) → ALORS un brick = double charge : ne pas l'empiler sur une semaine déjà saturée.`
- **Erreurs :** trop de bricks / trop durs (fatigue) ; brick à des intensités qui ne sont pas celles de la course (inutile) ; négliger les transitions ; tester le brick long sans nutrition.

---

## 6. EXEMPLES
- **Sprint/Olympique (spécifique) :** 60-75 min vélo dont blocs proche seuil → 15-20 min course @ allure course rapide. + répétitions T2.
- **70.3 (spécifique) :** 2h30-3h vélo @ IF 0,80-0,85 → 30-40 min course @ allure semi, avec ravito.
- **Ironman (spécifique) :** 4-5 h vélo @ IF 0,68-0,72 → 30-45 min course @ allure marathon, **plan nutrition complet** (B9), test matériel.

---
