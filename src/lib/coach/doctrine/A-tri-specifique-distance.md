# A — TRIATHLON · SPÉCIFIQUE DISTANCE (Sprint / Olympique / 70.3 / Ironman)

> **Statut : DOCUMENT MÉTHODE (spécificité d'épreuve).** Exigences détaillées par format = **B6 §4** ; architecture = **A-tri-periodisation-multisport** ; enchaînements = **A-tri-brick** ; pacing/zones = **B4** ; nutrition = **B9** (déterminante sur le long) ; affûtage = **B5 §7**.
> **Mapping charge :** IF vélo / allures (B4, B6) → SM/SN 〔HYP〕. **Plus le format est long, plus la NUTRITION et le PACING dominent le moteur (B6 §4).**

---

## 1. PRINCIPE
La préparation se **spécialise selon la durée** de l'épreuve. La règle universelle (B6 §0) : intensité soutenable ∝ 1/durée → le **pacing vélo cible (IF)** baisse et le **poids de la nutrition** monte avec la distance. La discipline reine du triathlon longue distance n'est pas la puissance, c'est **rouler assez prudemment pour bien courir** et **ne pas se tromper de ventre**.

---

## 2. SPRINT (≈ 750 m / 20 km / 5 km — ~1h00-1h20)
- **Demande (B6 §4) :** proche seuil tout du long ; **transitions** = enjeu majeur (proportion du temps total).
- **Pacing :** nage maîtrisée (ne pas se mettre dans le rouge dès l'eau), vélo proche seuil, course rapide.
- **Entraînement :** seuil/VO2 dans les 3 sports, bricks **courts et intensifs** (A-tri-brick), répétition T1/T2.
- **Nutrition :** quasi nulle (B9) — glycogène plein au départ suffit.
> `SI sprint → ALORS travailler la tolérance à l'intensité soutenue + les transitions ; la nutrition n'est pas un sujet.`

## 3. OLYMPIQUE (≈ 1,5 / 40 / 10 — ~1h50-2h45)
- **Demande :** seuil soutenu + courir vite après le vélo.
- **Pacing :** vélo ~90-95 % FTP, course « négative » si possible.
- **Entraînement :** seuil prioritaire, VO2 d'entretien, bricks à allure course, nage seuil/eau libre.
- **Nutrition :** légère (gel/boisson si > 90 min).

## 4. 70.3 (≈ 1,9 / 90 / 21,1 — ~4h-6h30)
- **Demande (B6 §4) :** tempo/sub-seuil ; **discipline de pacing vélo + carburant + course off-bike**.
- **Pacing :** **vélo IF ~0,80-0,85 FTP** (rouler « trop facile » pour pouvoir courir) ; course @ allure semi cible, régulière.
- **Entraînement :** gros volume tempo/seuil vélo (faible SN, B5 §8), allure course (A-run-allure-specifique), **bricks longs** (A-tri-brick), nage CSS/eau libre.
- **Nutrition (B9) :** **60-90 g glucides/h**, surtout à vélo (l'estomac court mal) ; hydratation/sodium ; **entraîner l'intestin**.
> `SI 70.3 → ALORS la séance reine = brick long (vélo IF cible → course allure cible) AVEC le plan nutrition. C'est là que la course se gagne ou se perd.`

## 5. IRONMAN (≈ 3,8 / 180 / 42,2 — ~8h-17h)
- **Demande (B6 §4) :** Z2 → bas Z3 ; **la NUTRITION est le facteur limitant n°1**, devant le moteur ; pacing, durabilité, chaleur.
- **Pacing :** **vélo IF ~0,65-0,72 FTP** (sortir du vélo « frais »), marathon régulier (négatif idéal). Une erreur d'IF vélo = marche au marathon.
- **Entraînement :** énorme volume Z2 (durabilité, B3 §2.2), tempo vélo, sorties longues vélo + course, **bricks longs** avec nutrition, nage continue/eau libre. Force d'endurance + prévention (B8) vu le volume.
- **Nutrition (B9) :** **80-90+ g glucides/h, sodium, fluide** ; charge glucidique ; plan répété à l'entraînement ; gestion chaleur (B4 §8 / B9).
- **Affûtage (B5 §7) :** long (12-21 j), TSB cible +15 à +25.
> `SI Ironman → ALORS dimensionner et ENTRAÎNER le plan nutrition comme une discipline à part entière. Un IM se perd plus souvent au ventre/au pacing qu'aux jambes (B6 §4).`
> `SI chaleur attendue le jour J → ALORS réviser l'IF vélo et l'allure course à la baisse, augmenter sodium/fluide, accepter une tolérance digestive réduite (B9 §8).`

## 6. PÉRIODISATION SPÉCIFIQUE (B5)
- **Build → Spécifique :** converger vers les **intensités exactes du format** (IF vélo, allure course) et la **durée** (sorties/bricks de plus en plus représentatifs).
- **Répétition de course :** simuler pacing + nutrition + matériel + transitions sur les bricks longs.
- **Affûtage :** selon le format (B5 §7 / B6 §4), maintenir la fréquence des 3 sports.

---

## 7. ERREURS / CAS LIMITES (toutes distances)
- **Rouler trop fort pour le format** (IF trop haut) → course sabotée. L'erreur n°1 du triathlon longue distance.
- **Nutrition non entraînée** (70.3/IM) → troubles digestifs / mur (B9).
- **Négliger les transitions** (formats courts) → temps perdu gratuitement.
- **Nager trop fort au départ** → dette d'oxygène qui plombe vélo + course.
- **Sous-estimer la chaleur/durabilité** sur le long.
